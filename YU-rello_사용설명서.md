# YU-rello 사용설명서

YU-rello는 Trello와 유사한 칸반 보드 기반의 프로젝트 관리 도구입니다.
리스트와 카드를 사용해 팀의 업무를 시각적으로 관리할 수 있습니다.

## 1. 계정 만들기 / 로그인

### 회원가입
1. `/signup` 페이지 접속
2. **이름**, **이메일**, **비밀번호** (6자 이상) 입력 후 [회원가입] 클릭
3. 입력한 이메일로 인증 메일이 발송됩니다
4. 이메일 받은편지함에서 인증 링크를 클릭
5. `/login` 페이지에서 로그인

### 로그인
- 이메일과 비밀번호 입력 후 [로그인] 클릭
- 로그인 성공 시 대시보드로 이동

### 로그아웃
- 화면 상단 헤더의 [로그아웃] 버튼 클릭

## 2. 대시보드 - 보드 관리

대시보드는 내가 속한 모든 보드를 카드 형태로 보여주는 홈 화면입니다.

### 새 보드 만들기
1. [새 보드 만들기] 버튼 클릭
2. 보드 이름 입력 후 [만들기] 클릭
3. 생성된 보드로 자동 이동

### 보드 삭제
- 보드 카드에 마우스를 올리면 우측 상단에 **⋯** 버튼이 나타납니다
- 클릭 후 [삭제] 선택 → 확인 시 삭제

> 보드를 삭제하면 해당 보드의 모든 리스트, 카드, 데이터가 함께 삭제됩니다.

## 3. 보드 화면 기본 사용법

보드 화면은 여러 **리스트**가 가로로 나열되고, 각 리스트 안에 **카드**들이 있는 구조입니다.

### 화면 구성
- **상단**: 보드 제목, 멤버 아바타, 멤버 초대 버튼 (소유자/관리자만)
- **툴바**: 뷰 전환(보드/캘린더), 🔔 알람, 멤버 필터
- **메인 영역**: 리스트 + 카드들

### 실시간 업데이트
다른 팀원이 카드나 리스트를 수정하면 **새로고침 없이 자동으로 화면이 갱신**됩니다.

## 4. 리스트 관리

### 리스트 추가
- 보드 가장 오른쪽의 **[+ 리스트 추가]** 버튼 클릭
- 리스트 이름 입력 후 [추가] 클릭

### 리스트 이름 변경
- 리스트 제목 텍스트를 클릭하면 편집 상태로 전환
- 수정 후 **Enter** 키로 저장, **Escape** 키로 취소

### 리스트 삭제
- 리스트 우측 상단 **⋯** 메뉴 클릭 → [삭제]
- 확인 시 리스트와 그 안의 모든 카드가 삭제됩니다

### 리스트 순서 변경
- 리스트 상단의 **드래그 핸들**을 클릭하고 끌어서 원하는 위치에 놓기

### 리스트 멤버 지정
- 리스트 헤더의 **+** 버튼 클릭 → 멤버 선택
- 지정된 멤버는 리스트 헤더에 아바타로 표시됩니다
- 멤버 필터 기능과 연동됩니다

---

## 5. 카드 관리

### 카드 추가
- 리스트 하단의 **[+ 카드 추가]** 버튼 클릭
- 카드 제목 입력 후 [추가] 클릭

### 카드 열기
- 카드를 클릭하면 상세 모달이 열립니다

### 카드 순서/위치 변경
- 카드를 드래그하여 같은 리스트 내에서 순서 변경
- 다른 리스트로 드래그하여 이동 가능

### 카드 우클릭 메뉴
카드를 **우클릭**하면 메뉴가 나타납니다:
- **카드 복사**: 다른 리스트나 위치에 카드 복사
- **카드 삭제**: 확인 후 삭제

### 카드에 표시되는 정보
카드에는 다음 정보가 한눈에 표시됩니다:
- 라벨 (색상 바)
- 체크리스트 진행률 (✓ 완료/전체)
- 댓글 수
- 마감일
- 담당 멤버 아바타

---

## 6. 카드 상세 - 모달 기능

카드를 클릭하면 상세 모달이 열립니다. 모달 밖을 클릭하거나 **X** 버튼으로 닫을 수 있습니다.

### 제목 변경
- 모달 상단의 제목 클릭 → 편집
- **Enter** 저장 / **Escape** 취소

### 설명 작성
- [설명 편집] 클릭 → 텍스트 입력
- [저장] 또는 [취소] 버튼으로 완료

---

## 7. 체크리스트

카드 안에 여러 개의 체크리스트를 추가할 수 있습니다.

### 체크리스트 추가
- 카드 모달에서 [체크리스트 추가] 버튼 클릭
- 이름 입력 (기본값: "체크리스트")
- 다른 체크리스트에서 항목 복사 옵션 제공

### 체크리스트 항목 추가
- [+ 항목 추가] 클릭
- 항목 이름과 마감일(선택) 입력 후 [추가]

### 항목 완료 처리
- 항목 왼쪽 체크박스 클릭 → 완료/미완료 토글
- 완료된 항목은 취소선으로 표시
- 체크리스트가 100% 완료되면 진행 바가 초록색으로 표시

### 항목 순서 변경
- 항목의 드래그 핸들(⠿)을 드래그하여 순서 변경

### 항목 편집/삭제
- 항목을 클릭하면 편집 모드
- 제목과 마감일 수정 후 [저장]
- 항목 우측 [삭제] 버튼으로 삭제

### 체크리스트 이름 변경 / 삭제
- 체크리스트 제목 클릭 → 편집 (Enter 저장)
- [삭제] 버튼으로 체크리스트 전체 삭제

---

## 8. 라벨

카드에 색상 라벨을 붙여 분류할 수 있습니다.

### 라벨 추가
- 카드 모달의 라벨 섹션에서 [라벨 추가] 클릭
- 기존 보드 라벨 중 선택하거나 새 라벨 생성

### 새 라벨 만들기
1. 라벨 피커에서 [새 라벨 만들기] 섹션 펼치기
2. 라벨 이름 입력
3. 색상 선택 (빨강, 주황, 노랑, 초록, 파랑, 남색, 보라, 핑크)
4. [만들기] 클릭 → 자동으로 카드에 부착

### 라벨 제거
- 라벨 피커에서 체크된 라벨을 다시 클릭하면 제거

---

## 9. 멤버 관리

### 보드 멤버 초대 (소유자/관리자만)
- 보드 상단 [멤버 초대] 버튼 클릭
- 사용자 목록에서 추가할 멤버 선택
- [제거] 버튼으로 멤버 제거 가능

### 카드 담당자 지정
- 카드 모달의 멤버 섹션에서 [멤버 추가] 클릭
- 보드 멤버 중 선택 → 카드에 담당자로 등록
- 담당자로 지정된 멤버는 카드 활동 시 알림을 받습니다

### 리스트 멤버 지정
- 리스트 헤더의 **+** 버튼 클릭
- 멤버를 선택하면 해당 리스트에 연결됩니다
- 멤버 필터 사용 시 해당 멤버가 속한 리스트의 카드가 필터링됩니다

---

## 10. 댓글

### 댓글 작성
- 카드 모달 댓글 섹션에서 텍스트 입력
- [등록] 버튼 클릭

### 댓글 편집/삭제
- 본인이 작성한 댓글에만 [수정] / [삭제] 버튼이 표시됩니다
- 수정 후 [저장] 클릭

### 댓글 알림
- 카드 담당자로 지정된 멤버들에게 댓글 등록 알림이 전송됩니다

---

## 11. 날짜 설정

카드와 체크리스트 항목에 날짜를 지정할 수 있습니다.

### 카드 날짜
- 카드 모달의 날짜 섹션에서 **시작일**과 **마감일** 설정
- 날짜 입력 후 자동 저장
- **X** 버튼으로 날짜 초기화

### 체크리스트 항목 날짜
- 항목 추가 또는 편집 시 마감일 입력 가능
- 캘린더 뷰에서 확인 가능

---

## 12. 캘린더 뷰

카드와 체크리스트 항목의 날짜를 달력 형태로 확인할 수 있습니다.

### 전환 방법
- 툴바의 **[캘린더 뷰]** 버튼 클릭

### 화면 구성
- **카드 이벤트**: 남색(인디고) - 카드의 시작일~마감일 표시
- **체크리스트 항목**: 완료 전 Navy 색, 완료 시 초록색
- 이벤트 클릭 시 해당 카드 모달이 열립니다

### 달력 탐색
- **이전/다음** 버튼으로 월 이동
- **오늘** 버튼으로 현재 월로 이동
- 월/주 뷰 전환 가능

---

## 13. 알림 (Notification)

### 알림을 받는 경우
- 카드의 담당자로 지정되었을 때
- 내가 담당자로 있는 카드에 댓글이 달렸을 때
- 보드 멤버로 초대받았을 때

### 알림 확인
- 헤더의 🔔 아이콘 클릭 → 최근 알림 10개 표시
- 읽지 않은 알림은 파란 배경으로 표시
- 알림 클릭 → 해당 보드로 이동 + 읽음 처리
- [모두 읽음] 버튼으로 전체 읽음 처리

---

## 14. 예정 업무 알람 (🔔)

보드 화면 툴바의 🔔 버튼을 통해 **이 보드에서 오늘 이후 마감인 체크리스트 항목**을 확인할 수 있습니다.

- 클릭 시 상위 5개의 예정 항목이 드롭다운으로 표시
- 항목명, 마감일, 담당 멤버 정보 확인 가능

> 헤더의 알림 벨(NotificationBell)과는 별개입니다.

---

## 15. 카드 복사

기존 카드를 다른 리스트나 위치에 복사할 수 있습니다.

### 복사 방법
1. 카드 우클릭 → [카드 복사] 또는 카드 모달 하단 [카드 복사] 버튼
2. **제목** 수정 (선택)
3. **체크리스트 포함 여부** 선택 (체크리스트가 있는 경우)
4. **대상 리스트** 선택
5. **위치 (몇 번째)** 선택
6. [복사] 클릭

---

## 16. 멤버 필터

특정 멤버의 카드만 보고 싶을 때 사용합니다.

- 툴바 오른쪽 **멤버 드롭다운** 클릭
- 멤버 선택 → 해당 멤버가 담당자로 지정된 카드만 표시
- [전체] 선택 시 필터 해제

---

## 17. 보드 패닝 (화면 이동)

리스트가 많아 화면을 벗어날 때 마우스로 보드를 이동할 수 있습니다.

- 리스트/카드/버튼이 없는 **빈 영역**에서 클릭 후 드래그
- 가로/세로 방향으로 보드 이동 가능

## 권한 안내

| 역할 | 설명 | 주요 권한 |
|------|------|---------|
| 소유자(Owner) | 보드를 만든 사람 | 멤버 초대/제거, 모든 기능 |
| 관리자(Admin) | 시스템 관리자 | 멤버 초대/제거, 모든 기능 |
| 멤버(Member) | 보드에 초대된 사람 | 보드 열람, 카드/리스트 생성·수정·삭제 |
